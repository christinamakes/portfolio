Muscle Memory

Muscle Memory began with a need to track muscle group usage during my workouts. Sometimes I would complete a leg day only to realize I completly forgot my calves. Muscle Memory allows the user to create their own workout, using prebuilt or custom exercises, and track which muscle groups were used and what frequency they were used with during the workout.
