Muscle Memory

Project built in: 2 weeks

All innovation starts with frustration. Workout apps that use set routines worked well for me... until they didn't. I finished a leg day only to realize I completely forgot my calves! Muscle Memory allows the user to create their own workout by adding preset or custom exercises. The app tracks which muscle groups you use and how frequently you use them. This way, you can outsource the brains, and focus on the gainz.

Created using: React, Redux, MongoDB, Express, and Node

How does the news feel?

Project built in: 1 week

You know how you feel about the news, but how does the news feel about the news? With this app users can enter a newsworthy topic and get back an overall positive or negative score on that topic. This app uses News API to fetch relevant articles and Dandelion API to parse the emotion of every article.

Created using: Vue, Vuetify, Dandelion API, News API
